Adds voice search functionality to WordPress search bars using the Web Speech API.

== Description ==

Instant Voice Search adds voice recognition capabilities to your WordPress search forms, making it easier for users to search your website using their voice. This plugin is particularly useful for:

* Improving accessibility
* Mobile users who prefer voice input
* Users with disabilities
* Creating a modern, user-friendly search experience

= Features =

* Voice search integration with WordPress search forms
* Automatic language detection based on site language
* Works with default WordPress search and WooCommerce search
* Responsive design that works on all devices
* Full RTL (Right-to-Left) language support
* Zero configuration needed - works out of the box
* Lightweight and fast

= Browser Support =

The plugin uses the Web Speech API, which is supported in:
* Chrome (desktop and Android)
* Edge
* Safari (desktop and iOS)
* Opera

== Installation ==

1. Upload the `instant-voice-search` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! Voice search is automatically added to your search forms

== Frequently Asked Questions ==

= Which browsers support voice search? =

The plugin uses the Web Speech API, which is supported in:
* Chrome (desktop and Android)
* Edge
* Safari (desktop and iOS)
* Opera

= Does this work with custom search forms? =

Yes, the plugin works with:
* Default WordPress search forms
* WooCommerce product search
